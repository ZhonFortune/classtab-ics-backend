# 项目简介 :page_with_curl:
本项目为 [ClassTab-ICS](https://github.com/ZhonFortune/classtab-ics) 的后端部分，用于处理用户数据、课程表数据等。

<br>

# 安装 :rocket: <a id="install"></a>
[使用教程](https://zhonfortune.github.io/classtab-ics/#/)

```shell
#安装
npm install -g cltab-ics-bend
```

## 运行

```shell
#后端
cltabi-bend
```

## 访问

```shell
#后端 (默认)
http://localhost:6058
```
